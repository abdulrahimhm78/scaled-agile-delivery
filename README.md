# Scaled Agile Delivery

Execution practices for coordinating delivery across multiple teams
and programs in enterprise environments.

## Purpose
To enable alignment, transparency, and predictable execution
across teams operating at scale.

## Coverage
- PI Planning and execution
- Cross-team dependency management
- Risk identification and escalation
- Program-level delivery visibility

## Intended Audience
- SAFe Scrum Masters
- Release Train Engineers
- Program and Delivery Managers
